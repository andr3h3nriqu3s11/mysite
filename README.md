# My website

This is the website for the [andr3h3nriqu3s.com](andr3h3nriqu3s.com)

## Developing

Run server for dev

```bash
yarn run dev

# or start the server and open the app in a new browser tab
yarn run dev -- --open
```

## Building

This will use the static adapter to create a static build of the website.

```bash
yarn run build
```
